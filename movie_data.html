Introduction¶

This data set contains information about 10,000 movies collected from The Movie Database (TMDb), including user ratings and revenue.

To start Analysis we need to Answer these questions:

  1. Question 1: What is The most type of movies produced?
  2. What are the highest-grossing movies?
  3. What are the movies that got the most votes average?
  4. Is there any relationship between vote average and runtime of movies?
  5. Is there any relation between popularity and runtime?
  6. Is there any relation between popularity and vote_avg?
  7. Question7: Is there any relation between budget and revenue?

# define function to plot scatter
def scatterVote(X,Y,Data,title):
    sns.lmplot(
    x=X,
    y=Y,
    data=Data,
    #scatter_kws={'color': 'darkgreen'},
    line_kws={'color': 'black'},
    );
    ax = plt.gca()

    ax.set_title(title)

    plt.show()
